# 🎓 EduQuiz App

An interactive quiz application built with **React**, **JavaScript**, **JSX**, **CSS**, **Firebase Authentication**, and **Firestore**.

---

## 📖 Description

This project is an educational quiz platform designed to showcase skills in modern frontend and backend web development. It allows teachers to create and manage quizzes, while students can view available quizzes, submit answers, and track their progress.

The application demonstrates proficiency in:

- ⚛️ React (JSX, components, hooks)
- 📜 JavaScript (ES6+)
- 🎨 CSS (modern layouts, responsive design)
- 🔐 Firebase Authentication (signup, login, role-based access)
- 🔥 Firestore Database (real‑time data, CRUD operations)
- 🧭 React Router (navigation and protected routes)

---

## ✨ Features

- **Landing Page**: A welcoming entry point with login/signup buttons and gradient background.
- **Authentication**: Secure signup and login using Firebase Authentication.
- **Role-Based Dashboards**: Separate interfaces for teachers and students.
- **Teacher Dashboard**:
  - Create quizzes (via Google Drive links)
  - View and delete quizzes
  - Inspect student submission data
- **Student Dashboard**:
  - Start quizzes
  - Submit answers
  - View submission status
- **Quiz Submission**:
  - Students submit their work through Google Drive links
  - Teachers can see submission timestamps and time taken
- **Responsive Design**: Clean, modern UI implemented with regular CSS classes.

---

## 🛠️ Tech Stack

- **Frontend**: React, JSX, CSS
- **Routing**: React Router DOM
- **Backend-as-a-Service**: Firebase (Authentication & Firestore)
